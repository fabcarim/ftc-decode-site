## Objective
Use clear, consistent names for devices in the Robot Controller config.

## Simple explanation
If names are predictable, your code is easier to read and reuse. Avoid spaces and keep a pattern.

## Code snippet
```java filename=ConfigExample.java
// Example names in the RC config:
// leftFront, rightFront, leftRear, rightRear
// armMotor, intakeMotor, liftMotor
```
